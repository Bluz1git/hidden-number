# Hidden Number Game

Một chiếc game giải trí giúp bạn kiểm tra nhân phẩm: Hệ thống sẽ chọn một số và giữ bí mật, nhiệm vụ của bạn là đoán xem nó là số mấy. Chúc bạn may mắn ( lần sau :)) )

---

A fun little game to test your luck: The system picks a secret number, and your task is to guess it. Good luck!
